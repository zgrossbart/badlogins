Bad Logins Alert

This prototype plots 216 distinct points of information over seven axis of data and makes it easy to see bad logins in a small time window. The small time window makes it possible to filter out uninteresting systems.

The baseline of successful logins are shown in gray and unsuccessful logins are shown in other colors. Each IP address is shown in its own color. As the same user name is tried multiple times the dots indicating the request move higher. This makes it easy and fast to differentiate hackers from valid users with invalid passwords and plot the trend to find security issues and system outages.

You can drill deeper into the data by moving your mouse over each dot.

This chart plots:

  1. Successful logins
  2. Unsuccessful logins
  3. Time
  4. IP address
  5. Username
  6. Username repititions
  7. System access
